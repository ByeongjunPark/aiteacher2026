# 🏫 AI·디지털 도구 활용 설계안 생성기 (KERIS 거시적 차원)

이 프로젝트는 **이해 중심 교육과정(UbD) 백워드 설계** 및 **T-M-A(전이-의미구성-습득) 모형**을 바탕으로 교사들의 거시적 차원 교수학습 설계안 작성을 돕는 웹 애플리케이션입니다.

작성한 정보는 교사들의 상태와 맥락 분석, TMA 정렬성을 분석하기 위해 **Upstage Solar Pro3** 모델을 거쳐 AI 피드백을 제공합니다.

---

## 🔑 Upstage API Key 설정 방법 (초보자 가이드)

프로젝트를 실행했을 때 AI 피드백이 정상적으로 작동하려면 Upstage API Key가 설정되어 있어야 합니다. 이 프로젝트는 **두 가지 방법**으로 API 키를 설정할 수 있습니다.

### 방법 1. 웹 화면(UI)에서 바로 설정하기 (코드 수정 불필요 - 추천!)
1. 웹 브라우저에서 이 설계안 생성기 페이지(`index.html`)를 엽니다.
2. 우측 상단에 있는 **`⚙️ API 설정`** 버튼을 클릭합니다.
3. 나타나는 입력창에 본인의 Upstage API Key(형식: `up_...`)를 붙여넣고 **`저장`**을 누릅니다.
4. 저장된 키는 브라우저 내부 저장소(localStorage)에 안전하게 기억되므로, 다음 번에 들어와도 다시 입력할 필요가 없습니다.

### 방법 2. HTML 소스 코드에서 기본값 변경하기
개발자이거나 매번 입력하기 번거로울 경우, 소스 코드에 직접 기본 키값을 주입할 수 있습니다.
1. `index.html` 파일을 텍스트 에디터(VS Code, 메모장 등)로 엽니다.
2. 아래와 같이 `DEFAULT_API_KEY` 부분을 검색합니다 (약 445번째 줄 부근).
   ```javascript
   // Default API Key (User provided)
   const DEFAULT_API_KEY = "본인의_API_키_여기에_입력";
   ```
3. 따옴표 안에 발급받은 Upstage API Key를 적고 파일을 저장하면 됩니다.

## 🚀 로컬 실행 방법
이 프로젝트는 빌드 과정이 필요 없는 순수 정적 HTML 파일입니다.
`index.html` 파일을 더블 클릭하여 웹 브라우저(Chrome, Edge 등)에서 바로 열어 사용할 수 있습니다.
